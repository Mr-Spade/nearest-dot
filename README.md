# nearest-dot

平面最近点对有关的一切。

> 使用vue，vite，electron等工具链构建，组件库源自element-plus。

## debug

要在本地debug，请确保您安装了node.js以及yarn，并在目录下运行：

```
yarn install
yarn run dev
```

随后，您可以根据提示打开浏览网页对结果进行预览。

## run

要对项目进行编译，请确保您安装了node.js以及yarn，并在目录下运行：

```
yarn install
yarn run build
yarn run make
```

随后，将会在out目录下生成与您的平台对应的可执行程序。
